## 2. サービス構成（顧客目的軸の2本柱＋相談枠）

技術ではなく「顧客が何をしたいか」で分類する。

### 柱1：イベント・体験AR
マルシェ・商業施設・展示会・地域イベント向け。関わりの深さ順に3段階で提供する。

| 形態 | 概要 |
|------|------|
| 既製レンタル | 完成済みコンテンツをそのまま貸出。URL納品、運営は主催者側 |
| カスタマイズレンタル | ロゴ・素材の差し替え。短時間で完了する開発体制が強み（**主力商品**） |
| 企画・出張サポート | 新規制作、当日の設営・運営同行まで |

代表的な体験：ARゲーム（例：Catch The Fruits!）、記念撮影フォトフレーム、フォトスポット。

### 柱2：印刷物・商品AR
名刺・チラシ・カタログ・パッケージ・CD向け。かざすと動画や3Dが飛び出す仕掛けで「配って終わりの紙」を体験に変える。

| プロダクト | 概要 |
|-----------|------|
| AR名刺 | 名刺から動画・3Dが飛び出す |
| ARチラシ／カタログ | 印刷物に映像体験を付加 |
| AR商品パッケージ／ラベル | 商品から動くコンテンツへ |
| AR CD（ARジャケット） | CDジャケット等が動き出す。仕組み・顧客目的が名刺・チラシと同じため柱2に含める |

### 相談枠（カスタム）
観光マップ・文化財演出・お店のオリジナル企画など、上記に当てはまらないもの。**独立した柱にはしない**（実績の薄さを目立たせないため）。**実績が1件できたら3本目の柱に昇格**させる方針。

※ARフォトフレームは柱1（イベント体験メニュー）が主だが、納品物として柱2にも登場し得る。

## 3. 料金（目安。確定値は `CLAUDE.md` 第7章）

| 分類 | 目安 |
|------|------|
| 印刷物・商品AR（名刺・チラシ等） | キャンペーン ¥15,000〜（正規 ¥30,000〜） |
| イベント・体験AR（レンタル） | 遠隔 キャンペーン ¥15,000〜（正規 ¥20,000〜・2026-07-02確定）／ 出張運営 ¥50,000〜 ＋カスタマイズ ¥30,000〜（レンタル費別） |
| オリジナル制作・カスタム相談 | ¥300,000〜 |

- **中心価格帯:** 出張運営やカスタマイズを含む依頼は ¥50,000〜¥150,000（本格的なオリジナル新規制作は ¥300,000〜）。
- **表示原則:** 正規価格を基準にし、キャンペーンは割引として従属表示（安価アンカーを防ぐ）。具体額は料金セクション／サービスページに一元化する。
- **【廃止】月額保守の3プラン（¥800 / ¥5,000 / ¥10,000）は廃止。**

## 4. 収益モデル

**制作費（フロー）＋ スポット・継続対応（ストック）のハイブリッド。**

### サポート設計（旧「6ヶ月無料保守」から変更）
- **動作保証:** 納品後1ヶ月（納品時点の主要ブラウザ・OSでの不具合修正）。
- **ホスティング（公開）:** 1年間は料金に含む。2年目以降の延長は有償（条件・金額は都度案内、2026-07-10確定）。1ヶ月経過後の動作は保証しない。
- **スポット対応:** 1ヶ月以降の不具合修正・内容変更・差し替えは都度見積り（¥5,000〜軽微／¥10,000〜差し替え）。

> WebARは端末のOS・ブラウザに依存し、将来の更新で表示が変わり得る。「一度作れば永遠に動く」という誤解を技術的事実にもとづいて防ぐための設計。

## 5. フェーズ別戦略

### フェーズ1: 実績構築（現在地）
- モニター／キャンペーン価格で心理的障壁を下げる（先着・事例公開協力を条件）。
- 実績公開OKを条件に、ポートフォリオの数と質を優先。
- 主力は「カスタマイズレンタル」。短時間で差し替え納品できる開発体制を強みにする。

### フェーズ2: 付加価値のマネタイズ
- 「制作物」単体 → 「体験設計・ディレクション」の費用化。

### フェーズ3: LTV最大化
- 出張運営・継続改修・複数イベント横断の関係づくり。
